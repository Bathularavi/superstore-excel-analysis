# 🛒 Superstore Sales Analysis — Excel

## 📌 Project Overview
End-to-end data analysis of the Superstore Sales dataset (9,994 rows, 21 columns) using Microsoft Excel. Covers data cleaning, statistical analysis, PivotTables, and an interactive KPI dashboard.

**Business Question:** Which products, regions, and customer segments should the Superstore target or avoid?

## 📊 Dataset
- Source: Kaggle Superstore Dataset (https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)
- Size: 9,994 rows x 21 columns | Period: Jan 2014 to Dec 2017
- Key columns: Sales, Profit, Discount, Category, Region, Segment, Ship Mode

## 🔍 Key Findings
| Finding | Detail |
|---------|--------|
| Total Sales | $2,297,201 |
| Total Profit | $286,397 (12.5% margin) |
| Loss-Making Orders | 18.7% of all orders |
| Worst Sub-Category | Tables (-$17,725 profit) |
| Best Category | Technology (17.4% margin) |
| Discount Impact | Orders >40% discount lose money 73% of the time |
| Peak Season | Q4 = 32-35% of annual revenue |

## 🗂️ File Structure
| Sheet | Contents |
|-------|----------|
| Raw Data | Original dataset untouched |
| Working Data | Cleaned + calculated columns |
| Statistical Analysis | Stats, outliers, correlation, hypotheses |
| Pivot Analysis | 5 PivotTables with slicers |
| KPI Dashboard | 6 KPIs with targets |
| Charts | 6 formatted business charts |

## 💡 Business Recommendations
1. Cap all discounts at 30% — orders above 40% consistently lose money
2. Review Tables sub-category — loss-making in all 4 regions
3. Invest in Technology — highest margin (17.4%) and growing
4. Address Q1 revenue gap with targeted promotions
